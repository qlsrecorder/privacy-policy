Privacy Policy

Data Collection

This extension records user interactions such as clicks, inputs, selections, and navigation steps in order to generate automated test scenarios.

The extension may process form inputs during recording for automation purposes.

We do not intentionally collect or request sensitive personal data such as passwords, credit card numbers, or personal identifiers.

If sensitive information is entered in a form during recording, it may be captured as part of the user’s testing scenario. Users are responsible for avoiding the recording of sensitive data.

Data Usage

  • Build automated test scenarios
  • Allow replay and export of user-defined test flows
  • Improve test automation accuracy within the application

Data Storage

All recorded data is stored locally in the user’s browser (localStorage / chrome.storage).

No data is transmitted to external servers unless the user explicitly triggers an export or upload action.

Data Sharing

We do not sell, rent, or share user data with third parties.

Data is only transmitted to the Qaleadoscope backend when the user explicitly initiates an export or upload.

Security

All data is handled locally and transmitted only over secure HTTPS connections when required by user action.
